# Movie Finder & Information System

A premium web-based movie discovery application built for the First Year End Semester Project.

## Project Status
- **Status 1**: UI/UX implemented with HTML5 and CSS3 (Flexbox/Grid).
- **Status 2**: Functional behavior with JavaScript and OMDb API integration.
- **Status 3**: DSA concepts (Sorting, Searching, Stacks, Queues) integrated into a dedicated sub-folder.

## Features
- **Dynamic Search**: Real-time movie data fetching using `async/await`.
- **DSA Integration**:
  - **Bubble Sort**: Custom sorting by Release Year (Newest/Oldest).
  - **Linear Search**: Custom filtering within local results.
  - **Queue**: "Watch Later" list following FIFO principles.
  - **Stack**: "Search History" tracking using LIFO.
- **Responsive Design**: Premium dark-mode interface with glassmorphism effects.

## Setup
1. `npm install`
2. `npm run dev`

## DSA Folder
The specific academic requirements for DSA are located in the `dsa/` directory.
- `dsa/dsa-logic.js`: Contains all custom data structures and algorithms.
